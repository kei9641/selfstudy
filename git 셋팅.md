## common work

__bash shell__

- git 연결 (1회)

  `$ git init`

  `$ git clone { github 주소 }`

- __무조건 할 거!!!!__ (pull)

  `$ git pull origin master`

- 브랜치 만들기

  `$ git checkout -b { branch 이름 }`

- 변경사항 push

  `$ git add .`

  `$ git commit -m "{ 변경 내용 }"`

  `$ git push origin { branch 이름 }`
