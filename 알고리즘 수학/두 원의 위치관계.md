# 두 원의 위치관계

- boj 1002 풀이 공식

  : 두 원의 위치 관계를 수학식으로 표현할 수 있어야 한다.

  ​                   **d < r - r' ≤ d** **≤ r + r' < d**

  

  1) 두 좌표가 같은 경우

  ​	반지름까지 같은 경우 : -1 (무한대)

  ​	반지름이 다른 경우 : 0개

  

  2) 두 좌표가 다른 경우

  ​	 **d < r - r'** or **r + r' < d** : 0개
   	**r - r' = d** or **d = r + r'** : 1개
   	**r - r' < d** **< r + r'** : 2개
